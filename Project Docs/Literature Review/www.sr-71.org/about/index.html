Although there is a heavy emphasis on the Blackbird family of aircraft, SR-71 Online: An Online Aircraft Museum is a web site dedicated to all of aviation. This web site contains a large image gallery, guides to selected aviation museums, fact sheets on more than a dozen modern military aircraft, and a Blackbird Archive, a comprehensive information guide on the A-12, YF-12, and SR-71. This section contains information about this project and those responsible for its creation.

Paul R. Kucher IV

This was supposed to be a personal web site when development started in 1998. However, it evolved into an information resource for aviation. This page provides some background information on the author of this web site.

SR-71 Online Contributors

This site has improved greatly through contributions of information, images, and web site corrections. Those who take the time to provide suggestions are critical to the success of this site and are truly thanked, but this page is dedicated to those who have supported this site to an extraordinary degree.

Guest Book

The guest book is where visitors openly comment on the web site or share their experiences with the Blackbird or other related aircraft. Please be sure to sign the guest book before you leave.

Awards:
