Questions 11-20 are general questions that deal with the SR-71. The challenge level ranges from easy to difficult.

  1. How was the SR-71C built?

    1. They ordered one from Northrop.
    2. The front of #17957 was mated to the rear of #06936.
    3. They re-designated a "B" model as the "C" model.
    4. The static test model was mated to the rear of #06934.

  2. How many SR-71Bs were built?

    1. 1
    2. 2
    3. 3
    4. 4

  3. What type of fuel does the SR-71 run on?

    4. JP-5

  4. How many SR-71As were built?

    2. 15
    3. 23
    4. 29
