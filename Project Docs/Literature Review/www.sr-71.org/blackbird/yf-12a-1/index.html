In July 2001, the SR-71 Flight Manual was posted to this web site. In keeping with that tradition, this site now hosts the 420-page YF-12A Utility Flight Manual. This electronic copy has been prepared to educate the public about the operation and flight characteristics of the YF-12A Blackbird aircraft. Every attempt has been made to make this presentation free of error. If you find any mistakes, particularly in the navigational structure, please notify me.
