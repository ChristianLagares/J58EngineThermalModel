Let the Blackbird adorn your computer screen with the following desktop wallpapers. The wallpapers are available in two sizes: 1024x768 and 1280x1024. Because there is no standard aspect ratio or resolution for computer screens, you will have to choose the appropriate resolution.

Note: 1024x768 is of aspect ratio 4:3, which means that it aliases to other resolutions that are also 4:3, including: 1600x1200, 1400x1050, 800x600, and 640x480. What does all of this mean? If you are running one of these resolutions, please download the 1024x768 version. Your computer's operating system will scale it appropriately. If in doubt, please download the 1024x768 version since it has the most common aspect ratio.
