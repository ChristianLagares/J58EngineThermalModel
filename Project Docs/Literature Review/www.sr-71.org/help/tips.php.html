This site works well with any standards compliant browser. However, there are some options in your browser that are only under your control. Here are a few tips for a better overall experience. Be sure to read this if you are having technical problems.

Be sure to have the latest version of Mozilla Firefox (1.0+), Konqueror (3.2+), Safari (2.0+), Mozilla Seamonkey (1.7+), Microsoft Internet Explorer (6.0+), or Netscape Navigator (6.2+).

This site officially endorses the use of Mozilla Firefox as your default browser.

Please use one of the browsers listed above. Also, if you can, please avoid the use of Microsoft Internet Explorer because it is not fully CSS compliant (through version 6). This site now relies on cascading style sheets for both formatting and layout. Future versions of this site will be completely semantic and may include CSS 2.1-compatible styles. Consequently, browsers that are uncapable of properly handling CSS may be banned in the future. As for now, however, browsers that do not properly handle CSS will be unable to render the formatting of this site. However, the content will still be accessible through graceful degradation.

Be sure to have JavaScript/ECMAScript enabled for viewing some web technologies found in this site. If you are a user of AOL 2.0 or less or a WebTV user, you will not be able to take advantage of JavaScript.

Be sure to install the latest version of QuickTime from Apple Computer for viewing some movie content. Linux users should check out Xine or MPlayer.

Be sure to have the font families Arial, Helvetica, or sans-serif on your machine. Most of this site is in Arial/Helvetica/sans-serif, at 10 point font. Most operating systems have at least one of these fonts. Also, be sure to have your text size set at medium and your font color at default.

Please maximize your screen. This site requires a minimum of 800 x 600 resolution. If you cannot fit the width of this site on your entire screen, your resolution is lower than this minimum requirement. Check your computer's control preferences for adjustments.
