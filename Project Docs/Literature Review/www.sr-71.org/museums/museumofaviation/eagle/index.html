Eagle Building

Home » Museums » Museum of Aviation » Eagle Building

Shaped like the "Star and Bars" insignia, the Eagle Building has two floors of displays that include both aircraft and artifacts and a third floor that contains both offices and a cafeteria. The main lobby displays the first production F-15 with a dramatic hangar scene mural. The second floor is home to the Georgia Aviation Hall of Fame.
