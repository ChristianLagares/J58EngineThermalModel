Linking is encouraged and if you would like to negotiate a link exchange, please contact me. Below are several banners officially endorsed by SR-71 Online for linking to this web site. If you prefer a text link, refer to the text link regulations.

Installation Instructions:

Please save the banner image of your choice to your computer for posting to a web server. Then copy and paste the source code below into your web page and modify it accordingly. I ask that you save the image to your web server for both faster loading and a decrease in the number of requests made to my own server. Besides, hot linking is banned on this site. If you need assistance, feel free to write to me.

Text Links

If you wish to add a text link to this site from your site, please do the following things: 1.) When you add the hyperlink, direct the link to http://www.sr-71.org/. 2.) Do not make the link misleading, please describe it as "SR-71 Online: An Online Aircraft Museum" or simply "SR-71 Online." The link may look like this:

SR-71 Online: An Online Aircraft Museum
